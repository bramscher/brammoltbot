## How to Export

### Option A: From Browser (Fastest)
1. Open `hdpm_avatar_backdrop_vertical.html` in a browser (Chrome, Safari, Firefox)
2. Right-click on the backdrop you want ‚Üí **Inspect** ‚Üí **DevTools**

### Option B: SVG to PNG/JPG (Professional)
Use any of these tools to convert `.svg` ‚Üí `.png` at high quality:

**Free Online:**
- [CloudConvert](https://cloudconvert.com/svg-to-png)
- [Online-Convert](https://image.online-convert.com/convert-to-png)

**Command Line (if you have imagemagick):**
```bash
convert -density 150 hdpm_avatar_backdrop_9-16.svg -quality 90 hdpm_avatar_backdrop_9-16.png
```

**Adobe/Professional:**
- Open in Illustrator, export as PNG @ 150% DPI for crisp text
- Open in Figma, export as high-res PNG

- **DaVinci Resolve:** Import SVG as video track background

## Layout Details

### Avatar Positioning
- **Avatar occupies:** Lower left 60% of frame (shown with dashed lines)
- **Safe area:** Entire left 2/3 of screen
- **Logo/Branding:** Top 20% (green bar)
- **Contact info:** Bottom right corner (always visible)
- **CTA:** Center-right (adjusts per format)

### Design Elements
- **Top Bar:** High Desert brand color (#4a9d6f) with name + subtitle
- **Avatar Zone:** Dashed border guides placement (60% lower left)
- **Contact:** Phone + website bottom right, white/light green text
- **CTA Text:** "Properties Available for Lease" centered right side
- **Glass Element:** Subtle top-right accent for modern feel
- **Background:** Soft gradient white-to-cream for professional conference room feel

## Using with Avatar Software

### Popular AI Avatar Platforms:

**HeyGen / Synthesia / D-ID:**
1. Create new video
2. Select "Custom background" or "Upload image"
3. Import PNG/SVG from these files
4. Position avatar in lower left (it'll align naturally with dashed zone)
5. Adjust avatar scale if needed (should cover ~60% of lower left)

**Runway / Pika:**
1. Upload SVG or PNG as background layer
2. Composite avatar on top in lower-left position
3. Set avatar opacity to 1.0, background to full

**CapCut (Mobile/Desktop):**
1. Import backdrop as first layer (60 sec clip or loop)
2. Add avatar video on top, position lower-left
3. Adjust scaling so avatar fills the dashed zone

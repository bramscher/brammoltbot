## The Rule: "If It's Not Blocked, It Doesn't Exist"

Don't trust your brain. Trust the calendar.

- Delegation review? Block it.
- Thinking time? Block it.
- Fly fishing? Block it or it won't happen.

---

## Whimsical Setup Instructions

1. **Create board**: "Brammo Weekly Plan"
2. **7 columns**: Mon-Fri + Weekend + Backlog
3. **Swim lanes** (horizontal rows):
   - Deep Work
   - Meetings
   - Delegation
   - Admin
   - Personal
4. **Cards**: Drag tasks into time slots
5. **Daily**: 5min morning drag-and-drop planning
6. **Weekly**: Sunday plan next week

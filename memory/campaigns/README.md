# Konmashi Campaign Specs

Three campaigns to be built in Konmashi for automated social media content generation.

| Campaign | Target | Goal |
|----------|--------|------|
| HDPM Vacancy Marketing | Renters in Central Oregon | Fill vacancies faster |
| Konmashi for PM Companies | Property management companies (Appfolio users) | Get Konmashi customers |
| Wellifi (Remy + MED) | Health-conscious consumers | Drive subscriptions |

## Output Channels (All Campaigns)
- Instagram (Reels + Carousels)
- TikTok (Short videos)
- YouTube (Shorts + longer content)

## Content Types
- AI Avatar videos (Heygen-style)
- Carousels (multi-image posts)
- Static posts with copy

See individual campaign files for detailed specs.
